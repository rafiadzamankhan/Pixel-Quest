# 🕹️ Pixel Quest

A fast-paced, retro-style arcade game built with **Vanilla JavaScript** and **HTML5 Canvas**.

Game Link: https://pixelquest1.vercel.app/ 

The game id optimized for both Desktop and Mobile browsers

-----

### 🚀 The Challenge

  * **The Goal:** Collect the golden orbs to gain **+10 points**.
  * **The Threat:** Avoid the red squares. Hitting one resets your position and costs you **-5 points**.
  * **The Leveling:** Every **100 points**, a new enemy is added to the screen.

### 🎮 How to Play

  * **On Desktop:** Move using `WASD` or `Arrow Keys`.
  * **On Mobile:** Use the custom-built, ergonomic touch D-pad directly in your phone's browser.

### 🛠️ Tech Stack

  * **Logic:** Vanilla JS (Physics, Collision Detection, Leveling)
  * **Graphics:** HTML5 Canvas API
  * **Styling:** CSS3 (Grid-based mobile controls & Neon theme)
  * **Deployment:** Vercel
